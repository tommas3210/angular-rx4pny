佈景主題

  • 應用開發自訂樣式位置 /implementation/style/index.less。 (預設樣式)
  • 所有自訂的主題樣式放置於 app/theme 資料夾, (/app/theme/theme-XXX.less)
    主題樣式命名規則：
    「theme-樣式名稱.less」
    Ex: theme-red.less (theme-紅色系.less)
  • 編譯產生的 css 放置於
    src/assets/themes/theme-樣式名稱.css
    Ex: src/assets/themes/theme-red.css
  • 整體的UI主色變數 @theme-color: 自訂顏色;
    下方列出常用的變數, 可快速變更基本畫面編排
  • 新增自訂樣式選擇器命名規則：
    「應用開發單位 - 類別/位置 - 功能/屬性」
    Ex: dap-btn-add (開發單位dap-按鈕-新增)

名稱 說明 類型 默认值 dwReference 提示文字 string - dwPlacement 位置有十二个方向topLeft, top, topRight, leftTop, left, leftBottom, rightTop, right, rightBottom, bottomLeft, bottom, bottomRight string bottom dwTrigger 三种触发方式click, hover, focus string click
